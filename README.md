# kanban
# Интерактивная Канбан-доска (Zero-Dependency Task Manager).
# Основные возможности:
# Управление задачами: Создание, перемещение между тремя этапами («К выполнению», «В работе», «Готово») и архивация.
# Drag-and-Drop: Интуитивный интерфейс — карточки можно перетаскивать мышкой между колонками.
# Архивация: Возможность временно скрыть задачи в «Архив» с функцией их мгновенного восстановления.
# Автономность: Работает как однофайловое приложение (HTML + JS), не требующее установки серверов или баз данных.
